Visão Geral do Sistema

  • Como posso começar a usar o sistema?

    Para começar a usar o sistema, você pode se inscrever para uma conta ou fazer login, se já tiver uma. Depois, explore o menu principal para acessar os diferentes recursos e funcionalidades disponíveis.

  • Quais são os principais recursos?

    O sistema oferece recursos para gerenciamento de transações financeiras, definição de metas financeiras, criação de categorias de despesas, geração de relatórios e entre outros.

  • Onde posso encontrar mais informações?

    Para obter mais informações sobre o sistema, consulte a seção de ajuda nesta tela ou entre em contato com nossa equipe de suporte para assistência adicional.

Navegação

  • Como navegar entre as diferentes telas?

    Use o menu principal na parte superior da tela para acessar as diferentes seções do sistema. Clique em cada opção para abrir as telas correspondentes.

  • O que cada opção do menu principal faz?

    Cada opção do menu principal oferece acesso a diferentes funcionalidades, como visualização de despesas, configurações de conta, criação de metas, entre outros. Passe o mouse sobre cada opção para ver uma breve descrição.

  • Posso voltar para telas anteriores?

    Sim, você pode voltar para telas anteriores clicando no botão “voltar” (←) do seu navegador ou usando a funcionalidade de navegação do sistema, se disponível.

Redefinição de senha

  • Como faço para recuperar minha senha?

    Para recuperar sua senha, vá para a tela de login e clique no link “Esqueceu a senha?”. Você receberá instruções sobre como redefinir sua senha por e-mail. Ou acesse as configurações de perfil do usuário e clique no botão “redefinir senha”.
